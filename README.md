# 🧠 Smart Route Planner

Interactive pathfinding visualizer built with React.
Simulates real-world route optimization using Dijkstra and A* algorithms.



## 🚀 Features

- 🔍 Dijkstra Algorithm visualization
- ⚡ A* Algorithm (optimized pathfinding)
- 🚧 Walls (obstacles)
- 🚦 Traffic simulation (weighted nodes)
- 🎯 Shortest path reconstruction
- 🎨 Clean UI with animations

---

## 🖥️ Demo

[https://smart-route-planner.vercel.app/]


## ⚙️ Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS (custom UI)

---

## 🧠 Algorithms

### Dijkstra

Guarantees shortest path by exploring all possible routes.

### A*

Uses heuristic to find the optimal path faster.
